# NetPulse

NetPulse is a real-time internet diagnostics dashboard built for continuous network monitoring, router alignment, latency analysis, and WiFi optimization.

Unlike traditional speed test websites, NetPulse continuously measures your network quality and visualizes it through a live responsive dashboard. This makes it easier to identify unstable connections, dead zones, latency spikes, and the best possible router placement.

---

# Features

## Continuous Monitoring

NetPulse continuously tracks:

* Download speed
* Upload speed
* Ping
* Jitter
* Packet loss
* Stability score

---

## Live Dashboard

The dashboard provides:

* Realtime animated charts
* Rolling network analytics
* Trend indicators
* Responsive layouts
* Modern dark-mode UI

---

## Alignment Mode

A fullscreen mode designed specifically for:

* Router positioning
* Antenna alignment
* Dead-zone testing
* WiFi optimization

Large live metrics and simplified visuals make it easier to move around while monitoring network quality in real time.

---

## Session Analytics

NetPulse helps you understand how your network behaves over time with:

* Peak and average speeds
* Latency consistency analysis
* Stability scoring
* Session summaries
* Historical performance tracking

---

## Export Support

Export session data using:

* CSV
* JSON
* PNG dashboard snapshots

# Why NetPulse?

Most speed test tools are designed for one-time benchmarking.

NetPulse is designed for continuous diagnostics and real-world optimization, helping users:

* Find the best router placement
* Detect unstable connections
* Analyze latency consistency
* Monitor network quality over time
* Improve overall connection reliability

The goal is not just higher Mbps numbers, but a more stable and usable internet experience.
